[2010 : 플러그](https://www.acmicpc.net/problem/2010)

## 풀이

새로운 멀티탭의 플러그 수에서 하나를 뺀 값을 더해주면 된다.

## 코드

```cpp
#include <stdio.h>
int main() {
	int n, i, k, sum = 1;
	scanf("%d",&n);
	for(i=0;i<n;i++) {
		scanf("%d",&k);
		sum += k-1;
	}
	printf("%d",sum);
	return 0;
}
```
