## 코드

```cpp
#include <stdio.h>
int i,j,c,a[5];
int main() {
	for (i=0;i<5;i++)scanf("%d",&a[i]);
	for (i=1;;i++){
		for(j=0,c=0;j<5;j++)if(i%a[j]==0)c++;
		if(c>=3)return ~printf("%d",i);
	}
	return 0;
}
```
