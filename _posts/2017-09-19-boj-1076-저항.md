[1076 : 저항](https://www.acmicpc.net/problem/1076)

## 풀이

잘 계산해주자 꺄르륵

## 코드

```cpp
#include <iostream>
#include <string>
using namespace std;

int i, j;
long long sum;
string a, b, c, s[] = {"black","brown","red","orange","yellow","green","blue","violet","grey","white" };

int main() {
	cin >> a >> b >> c;
	
	for (i = 0; i < 10; i++) {
		if (a == s[i]) sum += i * 10;
		if (b == s[i]) sum += i;
	}
	for (i = 0; i < 10; i++) if (c == s[i])
		for (j = 0; j < i; j++) sum *= 10;

	cout << sum;

	return 0;
}
```
