## 코드

```cpp
#include <stdio.h>
int main() {
    int t, h, w, n;
    scanf("%d", &t);
    while(t--) {
        scanf("%d %d %d", &h, &w, &n); n--;
        printf("%d%02d\n", n%h+1, n/h+1);
    }
    return 0;
}
```
