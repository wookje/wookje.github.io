[1476 : 날짜 계산](https://www.acmicpc.net/problem/1476)

## 풀이

역시 중국인은 똑똑해!!!

## 코드

```cpp
#include <stdio.h>
int m, s, e, x;
int main() {
	scanf("%d %d %d", &e, &s, &m);
	x = (e*6916+s*4845+m*4200)%7980;
	return ~printf("%d", x ? x : 7980);
}
```
