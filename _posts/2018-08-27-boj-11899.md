[11899 : 괄호 끼워넣기](https://www.acmicpc.net/problem/11899)  

## 풀이

왠지 그냥 짜면 돌아갈 것 같았어

## 코드

```cpp
#include <cstdio>
#include <stack>
using namespace std;

char s[55];
stack<char> stk;

int main() {
    scanf("%s", s);
    for (int i = 0; s[i]; i++) {
        if (!stk.empty() && stk.top() == '(' && s[i] == ')') stk.pop();
        else stk.push(s[i]);
    }
    printf("%d", stk.size());
    return 0;
}
```
