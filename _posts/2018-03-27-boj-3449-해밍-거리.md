[3449 : 해밍 거리](https://www.acmicpc.net/problem/3449)

## 풀이

끄아아아아

## 코드

```cpp
#include <iostream>
#include <string>
using namespace std;

int n;
string a, b;

int main() {
	cin.tie(0); ios_base::sync_with_stdio(0);

	cin >> n;
	while (n--) {
		int r = 0;
		cin >> a >> b;
		for (int i = 0; i < a.size(); i++) if (a[i] != b[i]) r++;
		cout << "Hamming distance is " << r << ".\n";
	}

	return 0;
}
```
