[1049 : 기타줄](https://www.acmicpc.net/problem/1049)

## 풀이

단가 후려치기 ㄱㄱ

## 코드

```cpp
#include <stdio.h>
int min(int a, int b) { return a < b ? a : b; }
int n, m, a = 1e9, b = 1e9, c, d;
int main() {
	scanf("%d %d", &n, &m);
	while (m--) {
		scanf("%d %d", &c, &d);
		a = min(a, c);
		b = min(b, d);
	}
	a = min(a, b*6);
	printf("%d", n/6*a + min(a, (n%6)*b));
	return 0;
}
```
