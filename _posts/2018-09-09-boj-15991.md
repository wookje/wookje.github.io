[15991 : 1, 2, 3 더하기 6](https://www.acmicpc.net/problem/15991)  

## 풀이

둠칫타칫

## 코드

```cpp
#include <cstdio>
const int m = 1e9+9;
int n, t, d[55555]={1,1,2,4};
int main() {
    for (int i = 4; i <= 55555; i++) {
        d[i] = ((long long)d[i-1]+d[i-2]+d[i-3]) % m;
    }
    for (scanf("%d", &t); t--;) {
        scanf("%d", &n);
        printf("%d\n", (d[n/2] + d[n/2-1]) % m);
    }
    return 0;
}
```
