[1449 : 수리공 항승](https://www.acmicpc.net/problem/1449)

## 풀이

앞에서부터 순서대로 덮어주면 된다

파이프 터진 순서가 정렬되어 있다는 얘기가 없으니 정렬도 꼭 해주자

## 코드

```cpp
#include <stdio.h>
#include <algorithm>

int n, l, a[1001];

int main() {
	scanf("%d %d", &n, &l);
	for (int i = 0; i < n; i++) scanf("%d", &a[i]);
	
	std::sort(a, a + n);
	
	int tmp = a[0], cnt = 1;
	for (int i = 0; i < n; i++)
		if (tmp + l - 1 < a[i]) tmp = a[i], cnt++;

	printf("%d", cnt);

	return 0;
}
```
