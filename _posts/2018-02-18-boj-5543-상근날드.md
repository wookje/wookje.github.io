[5543 : 상근날드](https://www.acmicpc.net/problem/5543)

## 풀이

조아조아

## 코드

```cpp
#include <stdio.h>
#include <algorithm>
int a, b, c, r, i = 2;
int main() {
    while (i--) scanf("%d %d %d", &a, &b, &c), r += std::min({ a,b,c });
    return ~printf("%d", r - 50);
}
```
