## 풀이

언젠가는 돌아갈게  
사랑할 자격 갖춘 나 되어

## 코드

```cpp
#include <cstdio>
int a, b, c, n;
int main() {
    scanf("%d %d %d", &a, &b, &n);
    while (n--) a %= b, a *= 10, c = a / b;
    printf("%d", c);
    return 0;
}
```
