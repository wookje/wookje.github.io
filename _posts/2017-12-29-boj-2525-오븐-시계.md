[2525 : 오븐 시계](https://www.acmicpc.net/problem/2525)

## 풀이

나눗셈을 오지고 지리게 해보자.

## 코드

```cpp
#include <stdio.h>

int main() {
	int a, b, c;

	scanf("%d %d %d", &a, &b, &c);

	b += c, a += b / 60;
	b %= 60, a %= 24;

	printf("%d %d", a, b);

	return 0;
}
```
