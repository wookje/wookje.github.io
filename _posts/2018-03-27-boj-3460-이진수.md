## 풀이

## 코드

```cpp
#include <cstdio>
int main() {
	int tc;
	for (scanf("%d", &tc); tc; tc--) {
		int a, i = 0;
		scanf("%d", &a);
		while (a) {
			if (a & 1) printf("%d ", i);
			a >>= 1, i++;
		}
		puts("");
	}
	return 0;
}
```
