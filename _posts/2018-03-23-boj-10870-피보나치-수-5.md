[10870 : 피보나치 수 5](https://www.acmicpc.net/problem/10870)

## 풀이

엠티 가즈아~

## 코드

```cpp
#include <stdio.h>
int n, a, b = 1;
int main() {
	scanf("%d", &n);
	while (n--) {
		b += a;
		a = b - a;
	}
	printf("%d", a);
	return 0;
}
```
