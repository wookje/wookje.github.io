## 코드

```cpp
#include <cstdio>
int i, a, s, c[1001];
int main() {
    for (; i < 10; i++) scanf("%d", &a), s += a, c[a]++;
    for (; i < 1000; i += 10) if (c[c[0]] < c[i]) c[0] = i;
    printf("%d\n%d", s / 10, c[0]);
    return 0;
}
```
