[2522 : 별찍기 - 12](https://www.acmicpc.net/problem/2522)

## 풀이

오늘도 별이 바람에 스치운다

## 코드

```cpp
#include <cstdio>

int abs(int a) { return a < 0 ? -a : a; }

int main() {
	int n, i, j, k;
	scanf("%d", &n);

	for (i = 1; i < n * 2; i++) {
		for (j = 0; j < abs(n - i); j++) printf(" ");
		for (k = 0; k < n - j; k++) printf("*");
		puts("");
	}

	return 0;
}
```
