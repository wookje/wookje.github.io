[1377 : 버블 소트](https://www.acmicpc.net/problem/1377)

## 풀이

몇 번 움직였는지 세어보자

## 코드

```cpp
#include <cstdio>
#include <algorithm>
using namespace std;
int n, i, r;
pair<int, int> p[500005];
int main() {
    scanf("%d", &n);
    for (i = 0; i < n; i++) scanf("%d", &p[i].first), p[i].second = i;
    sort(p, p + n);
    for (i = 0; i < n; i++) r = max(r, p[i].second - i);
    printf("%d", r + 1);
    return 0;
}
```
