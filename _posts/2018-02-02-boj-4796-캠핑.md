## 풀이

룰루랄라

## 코드

```cpp
#include <stdio.h>
int a, b, c, d;
int main() {
	while (1) {
		scanf("%d %d %d", &a, &b, &c);
		if (!a && !b && !c) break;
		printf("Case %d: %d\n", ++d, c/b*a + (c%b < a ? c%b : a));
	}
	return 0;
}
```
