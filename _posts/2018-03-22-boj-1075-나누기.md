## 풀이

## 코드

```cpp
#include <stdio.h>
int a, b, i;
int main() {
	scanf("%d %d", &a, &b);
	a /= 100; a *= 100;
	for (; i <= 100; i++) if ((a + i) % b == 0) return ~printf("%02d", (a + i) % 100);
	return 0;
}
```
