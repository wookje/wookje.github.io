## 풀이

## 코드

```cpp
#include <stdio.h>
int n, a, b, c;
int main() {
    scanf("%d %d %d", &n, &a, &b);
    while (a != b) a = a/2+a%2, b = b/2+b%2, c++;
    printf("%d", c);
    return 0;
}
```
