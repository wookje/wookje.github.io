[2292 : 벌집](https://www.acmicpc.net/problem/2292)

## 풀이

`a[i]`를 중앙에서 i번째로 떨어진 껍데기를 이루는 육각형의 수라고 두면

`a[i] = a[i-1] + 6 (a[0] = 1, a[1] = 6)`이라는 식이 나온다.

이제 구현해보자!

룰루랄라

## 코드

```cpp
#include <stdio.h>
int main() {
	int n, i;
	scanf("%d", &n); n--;
	while (1) {
		n -= i*6, i++;
		if (n <= 0) break;
	}
	printf("%d", i);
	return 0;
}
```
