[1371 : 가장 많은 글자](https://www.acmicpc.net/problem/1371)

## 풀이

어디에도 그대가 살아서

## 코드

```cpp
#include <cstdio>
int ans, cnt[33];
char c;
int main() {
    while (~scanf("%c", &c)) cnt[c - 'a']++;
    for (c = 0; c < 26; c++) ans = ans > cnt[c] ? ans : cnt[c];
    for (c = 0; c < 26; c++) if (cnt[c] == ans) printf("%c", c + 'a');
    return 0;
}
```
