## 풀이

.

## 코드

```cpp
#include <cstdio>
int max(int a, int b) { return a > b ? a : b; }
int n, t, p, d[22];
int main() {
    scanf("%d", &n);
    for (int i = 0; i < n; i++) {
        scanf("%d %d", &t, &p);
        d[i + 1] = max(d[i + 1], d[i]);
        d[i + t] = max(d[i + t], d[i] + p);
    }
    printf("%d", d[n]);
    return 0;
}
```
