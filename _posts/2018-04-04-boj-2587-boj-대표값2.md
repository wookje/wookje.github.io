## 코드

```cpp
#include <cstdio>
#include <algorithm>
int i, s, a[5];
int main() {
    for (; i < 5; i++) scanf("%d", &a[i]), s += a[i];
    std::sort(a, a + 5);
    printf("%d\n%d", s / 5, a[2]);
    return 0;
}
```
