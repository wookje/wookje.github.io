[1225 : 이상한 곱셈](https://www.acmicpc.net/problem/1225)

## 풀이

눈누난

## 코드

```cpp
#include <cstdio>
long long i, s, r;
char a[10001], b[10001];
int main() {
	scanf("%s %s", a, b);
	for (i = 0; a[i]; i++) s += a[i] - '0';
	for (i = 0; b[i]; i++) r += s * (b[i] - '0');
	printf("%lld", r);
	return 0;
}
```
