[11728 : 배열 합치기](https://www.acmicpc.net/problem/11728)

## 풀이

조아조아조아조아조아

## 코드

```cpp
#include <cstdio>
#include <algorithm>
int a, b, c[2000001];
int main() {
	scanf("%d %d", &a, &b);
	a = b = a + b;
	while (a--) scanf("%d", &c[a]);
	std::sort(c, c + b);
	for (a = 0; a < b; a++) printf("%d ", c[a]);
	return 0;
}
```
