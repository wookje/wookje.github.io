[1302 : 베스트셀러](https://www.acmicpc.net/problem/1302)

## 풀이

다정했던 사람이여 나를 잊었나

벌써 나를 잊어버렸나

## 코드

```cpp
#include <iostream>
#include <string>
#include <map>
using namespace std;

int n, rn;
string s, rs;
map<string, int> m;

int main() {
	ios_base::sync_with_stdio(0); cin.tie(0);

	cin >> n;
	while (n--) cin >> s, m[s]++;

	for (auto it : m) if (rn < it.second)
		rn = it.second, rs = it.first;

	cout << rs;

	return 0;
}
```
