[1159 : 농구 경기](https://www.acmicpc.net/problem/1159)

## 풀이

사실 그냥 체킹 배열에 카운트 해줘도 되는데

map 안 쓴지가 오래돼서 써봤다 ㅎㅎ

## 코드

```cpp
#include <cstdio>
#include <map>
using namespace std;

int n;
char a[155];
map<char, int> s;

int main() {
	scanf("%d", &n);
	while (n--) scanf("%s", a), s[a[0]]++;
	for (auto it : s) if (it.second >= 5) putchar(it.first), n++;
	if (n == -1) printf("%s", "PREDAJA");
	return 0;
}
```
