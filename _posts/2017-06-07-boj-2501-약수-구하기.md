## 풀이

ㅎㅎ

## 코드

```cpp
#include <stdio.h>

int main() {
	int n, k, cnt=0;

	scanf("%d %d",&n,&k);

	for(int i=1;i<=n;i++) {
		if (!(n%i)) cnt++;
		if (cnt == k) {
			printf("%d", i);
			return 0;
		}	
	}
	puts("0");

	return 0;
}
```
