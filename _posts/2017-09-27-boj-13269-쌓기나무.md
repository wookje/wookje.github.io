[13560 : 축구게임](https://www.acmicpc.net/problem/13560)

## 풀이

랑주의 정리!

S[i] = a[1~i]의 합, 
`S[1~k] >= kC2`  
`S[n] = nC2`

*증명*



## 코드

```cpp
#include <stdio.h>
#include <algorithm>
int main() {
	int n, i, s = 0, a[10000];
	scanf("%d", &n);
	for (i = 0; i < n; i++) scanf("%d", &a[i]);
	std::sort(a, a + n);
	for (i = 0; i < n; i++) {
		s += a[i];
		if (s < (i+1)*i/2) return ~printf("-1");
	}
	printf("%d", s == n*(n-1)/2 ? 1 : -1);
	return 0;
}
```
