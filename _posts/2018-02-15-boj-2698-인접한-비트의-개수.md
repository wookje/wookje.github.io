[2698 : 인접한 비트의 개수](https://www.acmicpc.net/problem/2698)

## 풀이

그리워하면 언젠가 만나게 되는

## 코드

```cpp
#include <stdio.h>

int q, n, k, d[111][111][2];

int main() {
    d[1][0][0] = d[1][0][1] = 1;
    for (n = 2; n <= 100; n++)
        for (k = 0; k < n; k++)
            d[n][k][0] = d[n-1][k][0] + d[n-1][k][1],
            d[n][k][1] = d[n-1][k][0] + (k ? d[n-1][k-1][1] : 0);

    for (scanf("%d", &q); q--;)
        scanf("%d %d", &n, &k),
        printf("%d\n", d[n][k][0] + d[n][k][1]);

    return 0;
}
```
