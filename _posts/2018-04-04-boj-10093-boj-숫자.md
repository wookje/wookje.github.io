[10093 : 숫자](https://www.acmicpc.net/problem/10093)

## 풀이

알잖아 너를 이토록 사랑하며 기다린 나를  
뭐가 그리 바쁜지 너무 보기 힘들어  
넌 도대체 뭐하고 다니니

## 코드

```cpp
#include <cstdio>
#include <algorithm>
int main() {
    long long a, b;
    scanf("%lld %lld", &a, &b);
    if (a > b) std::swap(a, b);
    printf("%lld\n", b > a ? b - a - 1 : 0);
    for (++a; a < b; printf("%lld ", a++));
    return 0;
}
```
