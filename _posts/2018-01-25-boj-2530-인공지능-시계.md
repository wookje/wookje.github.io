[2530 : 인공지능 시계](https://www.acmicpc.net/problem/2530)

## 풀이

오.바.와.치

## 코드

```cpp
#include <stdio.h>
int main() {
	int a, b, c, d;
	scanf("%d %d %d %d", &a, &b, &c, &d);
	c += d;
	b += c / 60, c %= 60;
	a += b / 60, b %= 60;
	a %= 24;
	printf("%d %d %d", a, b, c);
	return 0;
}
```
