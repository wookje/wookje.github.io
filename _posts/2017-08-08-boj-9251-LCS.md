[9251 : LCS](https://www.acmicpc.net/problem/9251)

## 풀이

`dp[i][j] = a[1~i]와 b[1~j]의 LCS 값`

`dp[i][j] = max(d[i-1][j], d[i][j-1], d[i-1][j-1] + (a[i] == b[j]))`

## 코드

```cpp
#include <stdio.h>
#include <algorithm>

char a[1111], b[1111];
int i, j, d[1111][1111];

int main() {
	scanf("%s\n%s", a + 1, b + 1);

	for (i = 1; a[i]; i++) for (j = 1; b[j]; j++)
		d[i][j] = std::max({ d[i-1][j], d[i][j-1], d[i-1][j-1] + (a[i] == b[j]) });
	
	printf("%d", d[i-1][j-1]);

	return 0;
}
```
