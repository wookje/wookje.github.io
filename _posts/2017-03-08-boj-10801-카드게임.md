[10801 : 카드게임](https://www.acmicpc.net/problem/10801)

## 풀이

단순 구현.  

## 코드

```cpp
#include <bits/stdc++.h>
using namespace std;

int a[20], x, y;

int main() {
	for (int i = 0; i < 20; i++) scanf("%d", &a[i]);
	for (int i = 0; i < 10; i++) {
		if (a[i] > a[i + 10]) x++;
		if (a[i] < a[i + 10]) y++;
	}
	if (x > y) puts("A");
	else if (x < y) puts("B");
	else puts("D");

	return 0;
}
```
