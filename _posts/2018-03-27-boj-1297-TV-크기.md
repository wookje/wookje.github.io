[1297 : TV 크기](https://www.acmicpc.net/problem/1297)

## 풀이

수학 조아!!!

가로 세로 n:m의 비율이 주어지면 이 때의 대각선 비율을 d라고 하자.

이제 이걸 주어진 대각선 길이에 맞춰 나누고 곱하고 으쌰으쌰 해보자.

## 코드

```cpp
#include <cstdio>
#include <cmath>
int main() {
    int k, a, b;
    scanf("%d %d %d", &k, &a, &b);
    double d = k/sqrt(a*a+b*b);
    printf("%d %d", (int)(a*d), (int)(b*d));
    return 0;
}
```
