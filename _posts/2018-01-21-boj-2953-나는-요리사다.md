[2953 : 나는 요리사다](https://www.acmicpc.net/problem/2953)

## 풀이

아 출근하기 싫다  

## 코드

```cpp
i = 5, j, a, s, mn, mx;
main() {
	while (i--) {
		for (j = 0, s = 0; j < 4; j++) scanf("%d", &a), s += a;
		if (mx < s) mn = 5 - i, mx = s;
	}
	printf("%d %d", mn, mx);
}
```
