[1110 : 더하기 사이클](https://www.acmicpc.net/problem/1110)

## 풀이

do while 처음 써봤다 ㅋㅋ

## 코드

```cpp
#include <stdio.h>
int main() {
	int n, m, c = 0;
	scanf("%d", &n); m = n;
	do { c++, m = (m % 10) * 10 + (m / 10 + m % 10) % 10; } while (n != m);
	printf("%d", c);
	return 0;
}
```
