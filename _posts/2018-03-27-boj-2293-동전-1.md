[2293 : 동전 1](https://www.acmicpc.net/problem/2293)

## 풀이

`dp[i] = 동전 i원을 만들 수 있는 방법의 수`

`dp[i] = dp[i - coins[]]`

## 코드

```cpp
main() {
	int N, K, x, i, j;
	int dp[10001] = { 1, };
	scanf("%d %d", &N, &K);
	for (i = 1; i <= N; i++) {
		scanf("%d", &x);
		for (j = 0; j <= K - x; j++) dp[j + x] += dp[j];
	}
	printf("%d", dp[K]);
}
```
