[14726 : 신용카드 판별](https://www.acmicpc.net/problem/14726)

## 풀이

죽는 날까지 하늘을 우러러 한 점 부끄럼 없기를

## 코드

```cpp
#include <stdio.h>

int main() {
	int t;
	for (scanf("%d", &t); t--;) {
		int sum = 0;
		for (int i = 0, a; i < 16; i++) {
			scanf("%1d", &a);
			if (!(i % 2)) a *= 2;
			sum += a / 10 + a % 10;
		}
		if (sum % 10) puts("F");
		else puts("T");
	}

	return 0;
}
```
