[13458 : 시험 감독](https://www.acmicpc.net/problem/13458)

## 풀이

행복을 줄 수 없었어  
그런데 사랑을 했어  
네 곁에 감히 머무른  
내 욕심을 용서치마  

## 코드

```cpp
#include <stdio.h>

int n, i, a[1000001], b, c;
long long ans;

int main() {
	scanf("%d", &n);
	for (i = 0; i < n; i++) scanf("%d", a + i);
	scanf("%d %d", &b, &c);
	for (i = 0; i < n; i++) {
		ans++, a[i] -= b;
		if (a[i] > 0) ans += a[i] / c + (a[i] % c ? 1 : 0);
	}
	printf("%lld", ans);

	return 0;
}
```
