[2783 : 삼각 김밥](https://www.acmicpc.net/problem/2783)

## 풀이

그냥 짜면 된다!

## 코드

```cpp
#include <stdio.h>
double a, b, n, f;
int main() {
	scanf("%lf %lf %lf", &a, &b, &n);
	f = a / b;
	while (n--) {
		scanf("%lf %lf", &a, &b);
		f = f < a / b ? f : a / b;
	}
	printf("%lf", f * 1000);
	return 0;
}
```
