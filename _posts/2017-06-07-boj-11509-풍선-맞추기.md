[11509 : 풍선 맞추기](https://www.acmicpc.net/problem/11509)

## 풀이

높이 a의 풍선을 볼 때, a+1 높이에서 날아오는 화살이 있는지 검사해주자!

## 코드

```cpp
#include <iostream>
using namespace std;
int n, a, i, ans, cnt[1000001];
int main() {
	std::ios_base::sync_with_stdio(false);
	cin >> n;
	for (i = 0; i < n; i++) {
		cin >> a;
		if (!cnt[a+1]) cnt[a]++, ans++;
		else cnt[a+1]--, cnt[a]++;
	}
	cout << ans;
	return 0;
}
```
