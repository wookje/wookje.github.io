[2960 : 에라토스테네스의 체](https://www.acmicpc.net/problem/2960)  

## 풀이

하앙 나도 에라토스테네스처럼 고수이고 싶다

## 코드

```cpp
#include <cstdio>
int n, k, c, v[1001];
int main() {
	scanf("%d %d", &n, &k);
	for (int i = 2; i <= n; i++) {
		for (int j = 1; i*j <= n; j++) {
			if (!v[i*j]) c++;
			v[i*j] = 1;
			if (c == k) return !printf("%d\n", i*j);
		}
	}
    return 0;
}
```
