[2003 : 수들의 합 2](https://www.acmicpc.net/problem/2003)

## 풀이

투포인터 조아!!

## 코드

```cpp
#include <stdio.h>
int n, m, k, s, a[10001];
int main() {
	scanf("%d %d", &n, &m);
	for (int i = 0, j = 0; i < n; i++) {
		scanf("%d", &a[i]);
		s += a[i];
		while (s > m) s -= a[j++];
		k += (s == m);
	}
	printf("%d", k);
	return 0;
}
```
